# CodeAlpha_Stock_Portfolio_## 📌 Project Overview

This project is a **web-based Stock Portfolio Tracker** developed using Python and Flask.  
It allows users to input stock names and quantities, dynamically display them on a webpage, and calculate the total investment. The application also provides an option to save the portfolio data into a file for future reference.

---

## 🎯 Objective

The main objective of this project is to implement **basic programming and web development concepts** such as dictionaries, loops, conditional statements, and file handling, while also introducing backend integration using Flask in a practical and interactive way.

---

## 🚀 Features

- 📥 User input for stock names and quantities  
- 📊 Dynamic display of portfolio on the webpage  
- ➕ Ability to add multiple stocks continuously  
- 💰 Calculates investment for each stock  
- 💾 Save complete portfolio to a `.txt` file  
- ✅ Input validation for better user experience  
- 🎨 Clean and simple user interface
